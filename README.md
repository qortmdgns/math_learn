# Math Learning Platform

수학 학습을 위한 웹 기반 플랫폼입니다.

## 시작하기

1. 의존성 설치:
```bash
npm install
```

2. 개발 서버 실행:
```bash
npm run dev
```

3. 브라우저에서 [http://localhost:3000](http://localhost:3000)으로 접속하여 확인하세요.

## 기술 스택

- Next.js
- React
- Tailwind CSS

## 프로젝트 구조

```
math_learn/
├── components/     # 재사용 가능한 컴포넌트
├── pages/         # Next.js 페이지
├── styles/        # 전역 스타일
└── public/        # 정적 파일
```
